# Plugin Resources

- Shared plugin resources are stored in `resources/`.
- Example newsletters are stored in `resources/examples/`.
- RSoW example newsletters are stored in `resources/examples/rsow/`.
- BBtn example newsletters should be stored in `resources/examples/bbtn/` when available.
- `resources/examples/` is the development source for deriving compact style references.
- Skill-local `reference/` files should contain only excerpts or distilled examples that the skill actively needs.
- When adding examples, prefer updating the relevant skill-local reference file instead of making skills read full newsletters repeatedly.
- Keep bulky raw examples in `resources/`, not in `instructions/`.
