# Plugin Resources

- Shared plugin resources are stored in `resources/`.
- Example newsletters are stored in `resources/examples/`.
- RSoW example newsletters are stored in `resources/examples/rsow/`.
- BBtn example newsletters should be stored in `resources/examples/bbtn/` when available.
- Skills should treat `resources/examples/` as the canonical source for reusable example newsletters.
- Skill-local examples may exist only when they are specific to a single skill and should not replace the shared examples in `resources/examples/`.
- If useful for a skill, copy the relevant example newsletters, excerpts, or derived reference material from `resources/examples/` into that skill's folder.
- Copied skill-local material should live at the skill folder level or in a clearly named subfolder and should contain only what the skill needs.
